🧑‍🎓 Student Course Management System - C++

A **C++ console application** built using **Microsoft Visual Studio**.  
This project allows users to **manage students and their enrolled courses** through a command-based menu interface.

---

📖 📌 Project Overview
This project implements a **Student class** that represents a student with:

🔢 **Student ID**  
📛 **Name**  
📊 **GPA**  
📝 **List of enrolled courses**

The system allows:
- Adding new students  
- Removing existing students  
- Searching students by ID  
- Displaying all students  
- Enrolling students in courses  
- Showing student courses  
- Sorting students by GPA  

---

📌 Example Menu Commands
1 → Add Student
2 → Remove Student
3 → Search Student
4 → Display All Students
5 → Enroll Student in Course
6 → Show Student Courses
7 → Sort Students by GPA
8 → Exit
